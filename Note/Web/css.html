CSS

1. CSS 基础

1.1 CSS 是什么

  • 层叠样式表，用于描述 HTML 外观
  • 通过选择器匹配元素并应用规则
  • “层叠”包含继承、优先级与来源顺序

1.2 基本语法

1.3 引入方式

  • 外部样式：<link rel="stylesheet" href="style.css">
  • 内部样式：<style>...</style>
  • 行内样式：style="..."（不推荐）

2. 选择器

2.1 基础选择器

  • 通配：*
  • 类型：p、div
  • 类：.card
  • ID：#header

2.2 组合选择器

  • 后代：.nav a
  • 子代：.nav > a
  • 相邻兄弟：h1 + p
  • 通用兄弟：h1 ~ p

2.3 属性选择器

  • [type]、[type="text"]
  • [class^="btn"]、[class$="-primary"]、[class*="main"]

2.4 伪类与伪元素

  • 伪类：:hover、:focus、:active、:nth-child(n)、:not()
  • 伪元素：::before、::after、::first-letter、::placeholder

2.5 选择器优先级

  • !important > 行内 > ID > 类/属性/伪类 > 类型/伪元素

3. 层叠与继承

3.1 层叠规则

  • 同优先级时，后声明覆盖先声明
  • 来源顺序：作者样式 > 用户样式 > 浏览器默认样式

3.2 继承

  • 字体、颜色通常继承
  • 布局相关（如 margin、padding）不继承

4. 盒模型

4.1 组成

  • 内容 content
  • 内边距 padding
  • 边框 border
  • 外边距 margin

4.2 box-sizing

  • content-box（默认）
  • border-box（常用）

5. 显示与定位

5.1 display

  • block、inline、inline-block
  • flex、grid
  • none

5.2 position

  • static、relative、absolute、fixed、sticky
  • 配合 top/right/bottom/left

5.3 层叠上下文

  • z-index 仅对定位元素生效
  • 形成新层叠上下文的属性：position + z-index、opacity、transform

6. Flex 布局

6.1 容器属性

  • display: flex
  • 主轴：flex-direction
  • 换行：flex-wrap
  • 主轴对齐：justify-content
  • 交叉轴对齐：align-items、align-content

6.2 子项属性

  • flex: 1（缩写）
  • flex-grow、flex-shrink、flex-basis
  • align-self

7.1 容器定义

7.2 子项布局

  • line-height、letter-spacing

8.2 文本相关

  • text-align、text-decoration
  • text-transform、white-space
  • overflow-wrap、text-overflow

9.1 颜色表示

  • HEX：#ff0000
  • RGB/A：rgb(255,0,0)
  • HSL/A：hsl(0, 100%, 50%)

9.2 背景属性

  • background-color
  • background-image
  • background-size
  • background-position

  • aspect-ratio 保持比例

16. 常见问题

  • margin 折叠：父子或兄弟元素
  • 行内元素不支持 width/height
  • position: absolute 脱离文档流
  • z-index 需要定位上下文

17. 快速参考

  • 布局：display、position、flex、grid
  • 盒模型：margin、padding、border、box-sizing
  • 排版：font、text、line-height
  • 视觉：color、background、shadow
  • 动效：transition、animation

18. 值与单位体系

18.1 绝对与相对单位

  • 绝对单位：px、cm、mm、in、pt
  • 相对单位：em、rem、%、vw、vh、vmin、vmax

18.2 长度与角度

  • 长度：px、em、rem、%
  • 角度：deg、rad、turn

18.3 时间与频率

  • 时间：s、ms
  • 频率：Hz、kHz

18.4 颜色值

  • 十六进制：#RRGGBB、#RGB
  • rgb() / rgba()
  • hsl() / hsla()
  • 颜色函数：color-mix()（现代）

19. 级联、层与优先级

19.1 来源与重要性

  • 浏览器默认样式 < 用户样式 < 作者样式
  • !important 提升权重（谨慎使用）

19.2 级联层（@layer）

  • 层的顺序会影响最终规则

46. 常用属性详解（按模块）

46.1 盒模型与尺寸

  • width / height：设置内容盒尺寸；配合 box-sizing 决定计算方式
  • min-width / max-width：尺寸约束，防止过小或过大
  • min-height / max-height：高度限制
  • margin：外边距，可为负值；会发生外边距折叠
  • padding：内边距，不能为负值
  • box-sizing：content-box 或 border-box

46.2 边框与轮廓

  • border：简写 width style color
  • border-radius：圆角，可分别设置 4 角
  • outline：不占布局空间，常用于 focus
  • box-shadow：投影，inset 表示内阴影

46.3 显示与可见性

  • display：元素布局类型（block/inline/flex/grid/none）
  • visibility：visible / hidden（保留布局）
  • opacity：不透明度，0-1

46.4 定位与偏移

  • position：static/relative/absolute/fixed/sticky
  • top/right/bottom/left：定位偏移
  • z-index：层叠顺序（需定位上下文）

46.5 浮动与清除

  • float：left/right/none
  • clear：清除浮动影响

46.6 Flex 属性

  • 容器：flex-direction、flex-wrap、justify-content、align-items、align-content
  • 子项：flex、flex-grow、flex-shrink、flex-basis、order、align-self

46.7 Grid 属性

  • 容器：grid-template-columns、grid-template-rows、gap
  • 子项：grid-column、grid-row、grid-area
  • 自动：grid-auto-flow、grid-auto-columns、grid-auto-rows

46.8 文本与排版

  • font-family：字体栈
  • font-size：字号（可用 rem）
  • font-weight：字重（100-900）
  • line-height：行高，推荐无单位
  • text-align：对齐
  • text-decoration：下划线/删除线等
  • text-transform：大小写转换
  • letter-spacing：字间距
  • word-spacing：词间距
  • white-space：换行与空白处理
  • text-overflow：溢出显示（配合 overflow）

46.9 颜色与背景

  • color：文字颜色
  • background-color：背景色
  • background-image：背景图/渐变
  • background-size：背景尺寸
  • background-position：背景位置
  • background-repeat：背景平铺
  • background-attachment：滚动/固定

46.10 列表与表格

  • list-style：列表样式简写
  • list-style-type：符号
  • list-style-position：inside/outside
  • border-collapse：表格边框合并
  • border-spacing：表格间距

46.11 溢出与裁剪

  • overflow：溢出处理
  • overflow-x / overflow-y
  • clip-path：裁剪路径

46.12 变换与过渡

  • transform：位移/缩放/旋转
  • transform-origin：变换基点
  • transition：过渡简写
  • transition-duration：持续时间
  • transition-timing-function：缓动

46.13 动画

  • animation：动画简写
  • animation-name、animation-duration、animation-delay
  • animation-iteration-count、animation-fill-mode

46.14 交互与用户体验

  • cursor：鼠标样式
  • pointer-events：指针事件
  • user-select：文本选择
  • scroll-behavior：滚动行为

46.15 滤镜与混合

  • filter：滤镜
  • backdrop-filter：背景模糊
  • mix-blend-mode：混合模式

46.16 可访问性相关

  • outline：焦点提示
  • caret-color：光标颜色
  • accent-color：表单控件强调色
